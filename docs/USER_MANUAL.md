## 1. ข้อมูลทั่วไป

**SoilSensor** เป็นแอปพลิเคชันสำหรับใช้งานร่วมกับอุปกรณ์วิเคราะห์ดินแบบพกพา โดยมีความสามารถหลักดังนี้:

- เชื่อมต่ออุปกรณ์ผ่าน Bluetooth
- อ่านค่าจากเซ็นเซอร์แบบ Real-time
- บันทึกข้อมูลลงฐานข้อมูลพร้อมพิกัด GPS
- แสดงประวัติการวัดในรูปแบบกราฟและตาราง
- แสดงตำแหน่งการวัดบนแผนที่
- ให้คำแนะนำการปรับปรุงดิน
- ส่งออกข้อมูลเป็น Excel

### ค่าที่วัดได้

| ค่า | หน่วย | คำอธิบาย |
|-----|-------|---------|
| pH | - | ความเป็นกรด-ด่างของดิน |
| ไนโตรเจน (N) | mg/kg | ปริมาณไนโตรเจนในดิน |
| ฟอสฟอรัส (P) | mg/kg | ปริมาณฟอสฟอรัสในดิน |
| โพแทสเซียม (K) | mg/kg | ปริมาณโพแทสเซียมในดิน |
| ความชื้น | % | เปอร์เซ็นต์ความชื้น |
| อุณหภูมิ | °C | อุณหภูมิดิน |
| EC | dS/m | ความนำไฟฟ้า |
| ความเค็ม | ppt | ปริมาณความเค็ม (คำนวณจาก EC) |

## 2. ความต้องการของระบบ

### อุปกรณ์ที่รองรับ
- **Android:** 6.0 (Marshmallow) ขึ้นไป
- **iOS:** 13.0 ขึ้นไป

### สิทธิ์ที่ต้องอนุญาต
- **Bluetooth:** สำหรับเชื่อมต่ออุปกรณ์
- **Location/GPS:** สำหรับบันทึกพิกัดและแสดงบนแผนที่

### อุปกรณ์ที่จำเป็น
- อุปกรณ์ SoilSensor (พร้อมแบตเตอรี่)
- สมาร์ทโฟนที่รองรับ Bluetooth Low Energy

## 3. การติดตั้ง

### 3.1 ติดตั้งจาก App Store / Play Store
1. เปิด App Store (iOS) หรือ Play Store (Android)
2. ค้นหา "SoilSensor"
3. กด "ติดตั้ง" หรือ "Get"
4. รอให้ติดตั้งเสร็จ
5. เปิดแอป

### 3.2 อนุญาตสิทธิ์
เมื่อเปิดแอปครั้งแรก ระบบจะขอสิทธิ์:
- **Bluetooth:** กด "อนุญาต" เพื่อให้แอปสแกนและเชื่อมต่ออุปกรณ์
- **Location:** กด "อนุญาตขณะใช้แอป" เพื่อบันทึกพิกัด GPS

## 4. การใช้งาน

### 4.1 การเชื่อมต่ออุปกรณ์

1. **เปิดอุปกรณ์ SoilSensor**
   - เปิดสวิตช์อุปกรณ์
   - รอจอ LCD ติดและแสดงค่า

2. **เปิดหน้าสแกน**
   - กดไอคอน 🔍 "สแกน" ที่แถบนำทางด้านล่าง

3. **สแกนหาอุปกรณ์**
   - กดปุ่ม **"เริ่มสแกน"**
   - รอให้ระบบค้นหาอุปกรณ์ (ประมาณ 5-15 วินาที)

4. **เชื่อมต่อ**
   - เมื่อพบอุปกรณ์ จะแสดงชื่อ "SoilSensor"
   - กดปุ่ม **"เชื่อมต่อ"**
   - รอจนแสดงข้อความ "เชื่อมต่อสำเร็จ"

5. **ไปที่ Dashboard**
   - กด "ไปแดชบอร์ด" หรือกดไอคอน 🌱 "แดชบอร์ด"

> **💡 เคล็ดลับ:** หากไม่พบอุปกรณ์ ให้ตรวจสอบว่า:
> - อุปกรณ์เปิดสวิตช์แล้ว
> - Bluetooth บนโทรศัพท์เปิดอยู่
> - อุปกรณ์อยู่ใกล้โทรศัพท์ (ในระยะ 10 เมตร)

### 4.2 หน้า Dashboard

หน้านี้แสดงค่าเซ็นเซอร์แบบ Real-time จากอุปกรณ์

#### ส่วนประกอบ:
- **สถานะการเชื่อมต่อ:** แสดง "เชื่อมต่อแล้ว" (สีเขียว) หรือ "ไม่ได้เชื่อมต่อ" (สีแดง)
- **เวลาอัปเดต:** แสดงเวลาที่ได้รับข้อมูลล่าสุด
- **การ์ดค่าเซ็นเซอร์:** แสดงค่า pH, N, P, K, ความชื้น, อุณหภูมิ, EC, ความเค็ม

#### การใช้งาน:
- ดึงลงเพื่อ **รีเฟรช** ข้อมูล
- กด **"บันทึกผลการวัด"** เพื่อบันทึกข้อมูล
- กด **"ดูคำแนะนำ"** เพื่อดูการแนะนำการปรับปรุงดิน

### 4.3 การบันทึกผลการวัด

1. **เปิดหน้าบันทึก**
   - ที่หน้า Dashboard กดปุ่ม **"บันทึกผลการวัด"**

2. **เลือกชนิดพืช**
   - เลือกจากตัวเลือก: ข้าว, ข้าวโพด, มันสำปะหลัง, อ้อย, ยางพารา, อื่นๆ

3. **เลือกวิธีเก็บตัวอย่าง**
   - เก็บผิวดิน 0-15 cm
   - เก็บลึก 15-30 cm
   - เก็บลึก 30-60 cm

4. **เพิ่มหมายเหตุ (ถ้ามี)**
   - กรอกข้อมูลเพิ่มเติมในช่อง "หมายเหตุ"

5. **ตรวจสอบพิกัด GPS**
   - ระบบจะแสดงพิกัด GPS โดยอัตโนมัติ
   - หากไม่มีพิกัด ตรวจสอบว่าเปิด Location แล้ว

6. **บันทึก**
   - กดปุ่ม **"บันทึก"**
   - รอข้อความยืนยัน "บันทึกสำเร็จ"

### 4.4 ดูประวัติและกราฟ

1. **เปิดหน้าประวัติ**
   - กดไอคอน 📊 "ประวัติ" ที่แถบนำทาง

2. **กรองข้อมูล**
   - **ตามช่วงวัน:** เลือก 7 วัน / 30 วัน / 90 วัน
   - **ตามตำแหน่ง:** เลือกจาก dropdown (ถ้ามีข้อมูล GPS)

3. **สลับมุมมอง**
   - กด tab **"กราฟ"** เพื่อดูกราฟแนวโน้ม
   - กด tab **"ตาราง"** เพื่อดูรายการแบบตาราง

4. **ส่งออก Excel**
   - กดปุ่ม **"ส่งออก Excel"** ที่มุมขวาบน
   - เลือกวิธีแชร์ไฟล์ (อีเมล, ไลน์, ฯลฯ)

### 4.5 ดูแผนที่

1. **เปิดหน้าแผนที่**
   - กดไอคอน 🗺️ "แผนที่" ที่แถบนำทาง

2. **ดูจุดบนแผนที่**
   - Marker สีเขียว = จุดที่บันทึกไว้
   - กดที่ Marker เพื่อเลือกจุด

3. **ดูรายการด้านล่าง**
   - ดึงขึ้น/ลงเพื่อดูรายการจุดทั้งหมด
   - กดที่รายการเพื่อซูมไปยังจุดนั้น

4. **ล้างการเลือก**
   - กด **"ล้างการเลือก"** เพื่อยกเลิกการเลือกจุด

### 4.6 การตั้งค่า

1. **เปิดหน้าตั้งค่า**
   - กดไอคอน ⚙️ "ตั้งค่า" ที่แถบนำทาง

2. **สลับโหมดมืด**
   - เปิด/ปิด **"โหมดมืด"** เพื่อเปลี่ยนธีมสี

3. **ดูสถานะ BLE**
   - แสดงสถานะการเชื่อมต่อ
   - แสดงชื่ออุปกรณ์และ MAC Address (เมื่อเชื่อมต่อแล้ว)

4. **ตัดการเชื่อมต่อ**
   - กด **"ตัดการเชื่อมต่อ"** เพื่อยกเลิกการเชื่อมต่อ BLE

5. **ข้อมูลแอป**
   - แสดงเวอร์ชัน, Framework, ฐานข้อมูล, BLE Library

## 5. การแก้ไขปัญหา

### ปัญหา: ไม่พบอุปกรณ์
| สาเหตุ | วิธีแก้ไข |
|--------|----------|
| อุปกรณ์ปิดสวิตช์ | เปิดสวิตช์อุปกรณ์ |
| Bluetooth ปิดอยู่ | เปิด Bluetooth ในการตั้งค่าโทรศัพท์ |
| อยู่นอกระยะ | เข้าใกล้อุปกรณ์ (ในระยะ 10 เมตร) |
| อุปกรณ์เชื่อมต่อกับโทรศัพท์อื่น | ตัดการเชื่อมต่อจากโทรศัพท์เครื่องอื่นก่อน |

### ปัญหา: เชื่อมต่อแล้วตัด
| สาเหตุ | วิธีแก้ไข |
|--------|----------|
| แบตเตอรี่อุปกรณ์ต่ำ | ชาร์จแบตเตอรี่อุปกรณ์ |
| อยู่นอกระยะ | เข้าใกล้อุปกรณ์ |
| สัญญาณรบกวน | หลีกเลี่ยงพื้นที่มีสัญญาณแม่เหล็กแรง |

### ปัญหา: ไม่ได้พิกัด GPS
| สาเหตุ | วิธีแก้ไข |
|--------|----------|
| Location ปิดอยู่ | เปิด Location ในการตั้งค่าโทรศัพท์ |
| อยู่ในอาคาร | ออกไป outdoors หรือใกล้หน้าต่าง |
| สิทธิ์ไม่ได้รับอนุญาต | ไปที่ ตั้งค่า > แอป > SoilSensor > อนุญาต Location |

### ปัญหา: บันทึกไม่สำเร็จ
| สาเหตุ | วิธีแก้ไข |
|--------|----------|
| ไม่มี Internet | เชื่อมต่อ Wi-Fi หรือ Mobile Data |
| Server ปิดปรับปรุง | ลองใหม่ภายหลัง |
| ข้อมูลไม่ครบ | ตรวจสอบว่าเลือกชนิดพืชแล้ว |
